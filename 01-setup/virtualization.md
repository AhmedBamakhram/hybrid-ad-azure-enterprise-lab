# Virtualization Platform Decision

For this lab, we selected VMware Workstation Pro as the virtualization platform.

This lab is designed to simulate a real-world hybrid Active Directory and Microsoft Azure environment.

## Why VMware Workstation

- Better stability when running Windows Server environments  
- More reliable networking configuration for Active Directory labs  
- Improved performance and resource efficiency compared to alternative platforms  
- Robust snapshot functionality for easy rollback during testing  

## Version Used

We installed the latest available version, **25H2u1**, to ensure:

- Up-to-date features  
- Bug fixes and performance improvements  
- Maximum compatibility with modern Windows operating systems (Windows 10/11)  

## Summary

This setup ensures a stable and efficient environment for building and testing a Hybrid Active Directory and Azure lab, closely aligned with real-world enterprise infrastructure.
