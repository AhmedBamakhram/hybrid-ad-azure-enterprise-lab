## Lab Environment Setup

### Virtualization Platform
- Used VMware Workstation Pro (version 25H2u1)
- Selected for stability and reliable networking in lab environments

### System Requirements
- Host Machine: Windows 10/11
- RAM: Minimum 16 GB (recommended 32 GB)
- Storage: SSD with at least 100 GB free space

### Virtual Machines Created
- Domain Controller (Windows Server 2016)
- Client Machine (Windows 10/11)

### Network Configuration
- Configured internal virtual network
- Assigned static IP to Domain Controller
- Set DNS to point to Domain Controller

### Tools & Software
- VMware Workstation Pro
- Windows Server ISO
- Windows 10/11 ISO

### Validation
- Verified VM connectivity
- Confirmed client can communicate with Domain Controller
