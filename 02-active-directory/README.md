## Active Directory Configuration

### Environment Setup

* Installed Windows Server 2016 on a virtual machine
* Configured static IP address
* Set DNS to point to the Domain Controller

### Domain Controller Setup

* Installed Active Directory Domain Services (AD DS)
* Promoted server to Domain Controller (DC-01)
* Created domain: lab.local

### Organizational Units (OUs)

* Created OUs for:

  * IT
  * HR
  * Finance

### User and Group Management

* Created users for each department
* Created security groups
* Assigned permissions based on least privilege

### Testing

* Joined Windows 10 client to the domain
* Verified user login and authentication
